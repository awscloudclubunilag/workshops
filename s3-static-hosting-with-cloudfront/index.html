Welcome to our demo site! This page is hosted on Amazon S3 to show how easy and scalable static website hosting can be using AWS services.

AWS S3 provides secure, durable, and highly-scalable object storage. With just a few clicks, you can deploy your HTML, CSS, and JS files and serve them globally. This site was created by AWS Cloud Club UNILAG members to inspire students to build with the cloud.
